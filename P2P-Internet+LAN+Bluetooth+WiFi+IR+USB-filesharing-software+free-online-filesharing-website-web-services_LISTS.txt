Peer to peer file transfer tools
Peer to peer tools can be very useful for transfering large files due to their generally high or non-existent file size limits. This is due to these tools not requiring an intermediary server which uses lots of expensive network bandwidth.
The downside being that most of these tools require both parties to be online at the same time to use.
Find out more about how peer-to-peer file transfers work
