Best Answer:  Step 1: Select ALL columns of data you want sorted

Step 2: Under "Sort & Filter" is an option, 'Custom Sort', select that.

Step 3: Under 'Sort by' select Column 2, leave 'Sort On' as 'Value'
Under 'Order' select how you want the data to appear, ie: A-Z = Lowest to highest; Z-A = Highest to lowest

Step 4: Click OK, and Hey Presto! its done
